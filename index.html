What does javascript event loop do?

Converts JavaScript to machine language through the browser engine, as does the Chrome browser (V2 engine). It has two parts, one is (Heap) and the other is the call stack. The main function is in the call stack. Since JavaScript is a single threat language. That means they can only do one thing at a time. So when a function calls, one accumulates on top of the other like a stack. First the synchronous work is done and on the other hand the asynchronous work is done (Web ApI). Then I left the web api callback queue and then the event loop and work started. Event loop after a while to see if the mine function is free. If it is free then the callback queue is sent to the mine function one by one
